# Elysia with Bun runtime

## Getting Started

Paste this command into your terminal:

```bash
bun create elysia ./elysia-example
```

## Development

To start the development server run:

```bash
bun dev
```

Open http://localhost:3000/ with your browser to see the result.

## Prisma Client

```bash
bun prisma:generate
bun prisma:migrate
bun prisma:format
```
